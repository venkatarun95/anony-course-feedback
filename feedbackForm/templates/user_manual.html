User Manual

User Manual

The right pane shows the various things that you can do. In each of these the courses are listed department wise.

Analytics

The analytics tab will show overall as well as question-wise graphs along with basic first order and second order statistics.

Comments

You can view the comments for each courses, listed by question in the comments tab. Further a global word cloud is displayed displaying general trends in the comments. We plan to add sentiment-analysis and filters to remove the non-lexical words.

Private Messages

Private messages are entirely anonymous and private. The student can use this platform to discuss their concerns with the professor and to get counselling and support. This can greatly help improve the learning experience as students may be reluctant to come up in class. Obviously this tab can be accessed only by professors.

Create Forms

This is a powerful editor to edit the feedback form of the course. Both the instructors and the administrators can edit this. One can divide the form into sections and have two types of input. Rating based and text based. Rating based questions have a 5 point scale along with a 'Cannot Comment' option which is ignored while computing statistics.

Security

Here, we use a novel encryption system that enables previous feedbacks to be edited while still maintaining student anonymity. Further students can use the anonymity provided by this system to talk privately with the professor on matters which they are reluctant to discuss in person. We believe this will help address many student concerns as a professor's counselling and guidance can go a long way in improving a student's life in college. These messages may be shared with the administrator on the discretion of the professor.

All efforts have been taken to make this system as secure, reliable and user-friendly as possible. However if you face any issues or find anything lacking we would be glad to be informed of it.

Permissions Policy

Administrators: As the administrator, you have access to almost all the information about all the courses. However, private messages are, as the name suggests, private between the instructor and the student and feedback can only be given by students registered for the particular course.

Professors: Professors can access all relevant information about the courses they teach including the private messages.

Students: Students can give and modify their feedback anonymously. They can also view and initiate private conversations with the instructor in case they require such support.
